# SayIt
> Name idea from Yorushika (ヨルシカ) - "Say It." (言って。)

做這個工具是因為自己和 ChatGPT 聊英文的時候會詞窮，以及單字詞彙量不夠，加上特定的 Topic 時常不會講專業詞彙。因此做了一個簡單的工具來幫我即時記錄與提醒。

---

### 功能亮點
- **即時偵測**：每 3 秒自動掃描網頁上的使用者輸入內容。
- **重複警告**：當同一個詞出現超過門檻（3-5 次），立即跳出黑白極簡風面板提醒。
- **詞彙建議**：串接 Datamuse API，即時提供近義詞「小抄」，打破詞窮困境。
- **可移動 UI**：面板支援滑鼠拖動（Drag and Drop），可以隨意放置在不擋住對話框的位置。

### 安裝方法
1. 下載此專案至你的電腦。
2. 打開 Chrome 瀏覽器，進入 `chrome://extensions/`。
3. 開啟右上角的**「開發者模式」**。
4. 點擊**「載入解壓縮擴充功能」**，選取本專案資料夾即可完成安裝。
